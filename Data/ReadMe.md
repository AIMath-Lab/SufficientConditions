# The content of the CSV files

The files batchX_11_vertices_claw_free_claw_contractible_free.csv (X=0, ...,19) contains all claw-free and claw-contractible free graphs and their chromatic symmmetric functions. 

The file claw_free_stat_claw_contractible_10_vertices.csv contains data on all claw-free graphs, including their chromatic symmetric functions and an additional column indicating whether each graph is claw-contractible.
